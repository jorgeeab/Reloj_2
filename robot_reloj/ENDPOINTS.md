# Robot Reloj – API actual

Actualizado tras la migración a WebSockets (jul 2024). Este archivo resume los únicos puntos de entrada expuestos por `server_reloj.py`. Cualquier llamada fuera de esta lista es rechazada por el filtro `restrict_endpoints`.

## WebSockets

| Ruta | Descripción |
| --- | --- |
| `GET /ws/control` | Canal bidireccional para enviar setpoints, energías, ajustes PID, etc. Mensajes `{"type":"ping"}` reciben `{"type":"pong"}`. Cada payload válido produce un `control_ack` con el resultado de `apply_control_payload`. |
| `GET /ws/telemetry` | Telemetría continua. El servidor envía `telemetry_ready` y luego mensajes `{"type":"telemetry","status":{...}}` a intervalos definidos por `RELOJ_TELEMETRY_INTERVAL` (0.2 s por defecto). |

## HTTP

### UI / Meta

| Método | Ruta | Descripción |
| --- | --- | --- |
| GET | `/` | UI tradicional del robot. |
| GET | `/api` | Resume los endpoints activos y los canales WebSocket. |
| GET | `/api/ui/registry` | JSON con los widgets declarativos para construir UI dinámica. |

### Configuración y ajustes

| Método | Ruta | Descripción |
| --- | --- | --- |
| GET | `/api/config` | Datos mínimos para el hub (puerto serial, baudrate, estado). |
| GET | `/api/settings` | Carga el último `settings_ui.json`. |
| POST | `/api/settings` | Guarda ajustes persistentes. |

### Ejecución de tareas

| Método | Ruta | Descripción |
| --- | --- | --- |
| POST | `/api/tasks/execute` | Único endpoint para lanzar protocolos (`riego_basico` o `ir_posicion`). |
| GET | `/api/executions` | Lista ejecuciones activas y recientes. |
| GET | `/api/execution/<execution_id>` | Estado detallado de una ejecución. |
| POST | `/api/execution/<execution_id>/stop` | Solicita detener una ejecución (también detiene `protocolo_activo`). |

### Serial

| Método | Ruta | Descripción |
| --- | --- | --- |
| GET | `/api/serial/ports` | Enumera puertos disponibles y estado actual. |
| POST | `/api/serial/open` | Abre/reconfigura el puerto serial. |
| POST | `/api/serial/close` | Cierra el puerto serial. |

### Seguridad / control rápido

| Método | Ruta | Descripción |
| --- | --- | --- |
| POST | `/api/stop` | Apaga corredera, ángulo y bomba, y fuerza modo STOP. |
| POST | `/api/home` | Envía al robot a posición home (0,0). |
| POST | `/api/emergency_stop` | Parada de emergencia: corta energías y cierra el puerto. |
